📌 프로젝트 개요

본 프로젝트는 부업 실패 사례 데이터를 수집하고,
AI 모델 학습을 위한 구조화된 데이터셋을 구축하는 것을 목표로 한다.

데이터 수집 및 처리 과정:

네이버 지식iN API 기반 데이터 수집
실패 관련 키워드 필터링
경험담형 데이터 분리
상위 1000개 데이터 선별
데이터 클렌징 및 구조화

🧹 데이터 클렌징 기준

1. 텍스트 정리
HTML 태그 제거
특수문자 제거
공백 정리
2. 데이터 필터링
광고성 문구 제거
링크 및 홍보 텍스트 제거
중복 데이터 제거
3. 길이 기준
20자 미만 데이터 제거
🧠 데이터 구조

최종 데이터는 CSV 및 JSON 형태로 구조화하였다.

주요 컬럼
컬럼명	설명
source	데이터 출처 (naver_kin)
keyword	검색 키워드
title	글 제목
description	요약 내용
full_text	제목 + 본문
link	원문 링크
pubDate	작성일
text_length	텍스트 길이
label	라벨 (현재 unlabeled)
